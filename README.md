Технічне завдання: необхідно створити веб-сторінку, на якій буде форма з кнопкою Submit. 
На сторінці повинні бути наступні поля для вводу:
First name* (text)
Last name* (text)
Birthday* (date + datepicker)
Gender* (radio button)
Country* (select)
Email* (email)
Password* (password)
Address* (text)
Notes (textarea)  



На сторінці повинні бути виконані наступні вимоги:
При натисканні submit повинна відбуватись валідація даних, введених користувачем. 
Елементи з не валідними даними підсвічуються червоним кольором. Повідомлення з деталями помилки показуються під полем.
Валідація повинна відбуватись за допомогою дата атрибутів:
наприклад, якщо в елемента є data-validation=”email”, то валідними даними в цьому полі будуть тільки e-mail (використай регулярні вирази для email валідації)
* означає що поле обов’язкове для вводу. 
Текстові поля (text) дозволяють всі символи окрім ' та ".
При натисканні на поле birthday повинен відкриватись datepicker.
На кожен тип помилки повинно бути окреме повідомлення. Наприклад: якщо email поле пусте - “This field is required”. Якщо ж введено не валідний імейл - “Email address is not valid.”
Для виконання завдання можна використовувати jQuery UI.
Не використовуй плагіни для валідації даних.
При проходженні валідації покажи alert(‘Validation passed.’).
